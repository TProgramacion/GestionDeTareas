# Gestor de Tareas en C#

Aplicación de consola desarrollada que permite gestionar tareas mediante un sistema basado en un menú.  
Proyecto realizado en C#.

---

## Funcionalidades

- Crear tareas con:
  - Nombre
  - Descripción
  - Tipo (persona, trabajo, ocio)
  - Prioridad (alta/baja)
- Buscar tareas por tipo
- Eliminar tareas mediante ID
- Exportar tareas a fichero
- Importar tareas desde fichero

---

## Conceptos trabajados

- Variables y estructuras de control
- Listas
- Programación orientada a objetos
- Uso de `Enum`
- Lectura y escritura de ficheros

---

## Tecnologías utilizadas

- Lenguaje: C#
- Entorno: Visual Studio Code
